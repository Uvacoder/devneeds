What is whatdevsneed?

This website offers a list of tools for developers - ranging from paid software, to free open source projects. You might know some, but hopefully you'll discover new tools you've never seen. If you have a tool worth sharing, feel free to add it!

Why did you make this?

I wanted to share the list of tools I've bookmarked over the years. There might already be websites for this kind of thing, but I focused on listing everything from free to paid and picking out extra good tools and marking them as such (similar to curation).

Can I help?

Yes! The source code is open source. Check it out here! Maybe you can help me make new features, improve security or squash some bugs.
